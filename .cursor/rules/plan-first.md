# Plan First

Before editing any files:
- Create or update `PLANS.md` with goal, non-goals, acceptance criteria, approach,
  step plan, verification commands, and rollback plan.
- Then STOP and wait for the user to say "Execute".
- Do not change files until "Execute" is given.
